# road-accident-analysis-powerBI

🚦 Road Accident Analysis Dashboard (Power BI)

📌 Overview

🚀 This project is a Power BI dashboard designed to analyze road accidents across India and extract meaningful, actionable insights.

It demonstrates:

Real-world data analysis
Business intelligence skills
Data storytelling & visualization
🎯 Problem Statement

Road accidents are a major concern in India.
This project aims to answer:

👉 Where do accidents occur the most?
👉 What factors increase accident severity?
👉 How do time, weather, and road types influence risk?
